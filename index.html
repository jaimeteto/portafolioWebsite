PROJECTS

Wordle APP

This project is an Android-based version of the popular Wordle game, developed using Java and Android Studio. The app allows users to guess a five-letter word within six attempts, following the standard Wordle rules. After each guess, letters change color to indicate how close the player is to the correct word—green for correct letters in the right position, yellow for correct letters in the wrong position, and gray for incorrect letters.
